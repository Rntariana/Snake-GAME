#  Snake Game

A classic **Snake Game** built using HTML, CSS, and JavaScript. The player controls a snake that moves around the screen, eats food, grows longer, and tries to avoid crashing into walls or itself.

---

##  About the Game

The Snake Game is a simple yet addictive arcade-style game where the main goal is to eat as much food as possible to increase your score. Each time the snake eats food, it grows longer, making the game more challenging over time. The game ends if the snake collides with the wall or its own body.

This project is inspired by the classic arcade game :contentReference[oaicite:0]{index=0}, which became popular on early mobile phones and arcade machines.

---

## ✨ Features

- 🐍 Smooth snake movement
- 🍎 Random food generation
- 📈 Score tracking system
- 💥 Collision detection (walls & self)
- 🎮 Keyboard controls (Arrow keys / WASD)
- ⚡ Increasing difficulty as snake grows
- 📱 Responsive design

---

##  How to Play

1. Use arrow keys or WASD to move the snake  
2. Eat the food to grow longer and increase your score  
3. Avoid hitting the walls or your own tail  
4. Try to survive as long as possible and beat your high score
